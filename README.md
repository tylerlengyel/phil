# Phil

Fill your Phil. 

100% on-chain while being built on the original Ethereum. 

What is Phil? 

Phil is my first Generative Art Project being launched on Ethereum Classic. He's based on the 'Philosoraptor' Meme which I resonate with because I try to be a philosophizer myself here and there even though half the time idk what the fuck I'm talking about. Kind of like if the Dinosaurs came into the world we live in today trying to navigate it, Phil would feel lost, kind of like i was 3 years ago. So with that, the idea of Phil was first created.

How does it work?

Each trait is currently saved as an SVG, where the Path Data of each SVG is saved in a JSON file. When you visit each tab to 'Generate' a trait, there's an algorithm assigned to it, and there's a different one for each trait. The algorithm then pulls the Path Data from the JSON to ensure that the trait generation only happens within the traits dimensions, this way we don't color outside the lines :). Then once you find a trait image that you like, when you click 'Save', that traits SVG is then compressed into Hexadecimal format and saved for the Final Image which ultimately will be the NFT you mint.
Now, when you get to the Final Mint page, all of the hexadecimals will appear along with a preview of the final image. Once you click 'Mint NFT', the 9 different traits will then be stored as individual Ethscriptions, and each one will have a hash which is then sent to the Smart Contract and stored within the NFTs metadata. Therefor the token that you mint will have references to the 9 ethscriptions that you created making up your Phil. 

This way all data remains 100% on-chain and the only thing you need to render your image is a simple decompression script which I'll have open sourced along with the rest of the code after everything's been launched.

How many and how much? 

This is where things get a little different compared to most projects I've seen over the years. Phil is going to be split up into 3 phases.

The First Phase will be 420 Phils @ 0.42etc
*There will be a 69 hour cooldown before Phase 2 starts
The Second Phase will be 369 Phils @ 0.69etc
*There will be a 369 hour cooldown before Phase 3 starts
The Third Phase will be 69 Phils at 1.69etc

The cool thing here is that since there will be 3 different phases, it allows me to change the javascript files that are creating the images. Therefor the artwork for each Phase will be different. And who knows, it could even change during a phase...
